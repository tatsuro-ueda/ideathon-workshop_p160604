## 時間配分メモ

1. アイデア紹介＋全体の流れ（5分）
2. 他己紹介（10分）
  - ペアをつくる（2分）
  - 1人1分でお互いに自己紹介（3分）
  - 相手を他のペアに紹介（5分）
3. ブレスト（5分）
  - 1人目が書く（1分）
  - 2人目が書く（1分）
  - 3人目が書く（1分）
  - 4人目が書く（1分）
4. スピードストーミング（10分）
  - 1人目に話す（3分）
  - 2人目に話す（3分）
  - 3人目に話す（3分）
5. 評価（15分）
6. 優秀者発表（3分）
